# リステップ用户数据保存工具
[日本語バージョン](#日语)

一个用于リステップ的用户数据保存与迁移工具，用于在服务器关闭前对用户数据进行备份与导出。
在线版：https://backup.rst-game.top/


---

## 📌 项目简介（中文）

本项目是一个基于 Web 的用户数据保存与迁移工具，主要用于帮助用户在游戏服务器关闭前保存自身的账号数据与相关信息。

支持的功能包括：
- 用户基础数据读取与保存
- 历史活动排名数据抓取

本项目仅用于学习与技术研究用途，旨在展示 Web 环境下的数据请求、处理与保存方式，不用于任何商业用途。

---

## ⚠️ 注意事项（中文）

- 本工具仅用于学习与研究目的，不提供任何商业服务  
- 使用第三方工具可能违反游戏服务条款，存在账号限制或封禁风险，请自行承担后果  
- 本网站仅供 Android 用户使用，iOS 设备可能导致数据异常或丢失  
---

## 🧩 功能说明（中文）

- **迁移数据**：保存用户基础信息与部分游戏数据  
- **活动排名抓取**：支持历史活动排名数据获取（耗时较长）  

---

## 🚀 使用方式（中文）

1. 输入用户 ID（player_id）
2. 输入继承码（handover_code）
3. 选择功能按钮：
   - 左侧：快速保存基础数据
   - 右侧：抓取完整活动排名数据
4. 等待任务执行完成并查看结果
